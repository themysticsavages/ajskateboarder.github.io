Commands

All the commands Cloudbot has to offer (as of now)

    Utilities
    Command       Description
    weather        Get today's forecast in a city
    search        Get the first result of a Yahoo search
    youtube        Get the first five videos from a keyword
    shorten        Shorten a URL with Rebrandly
    topmeme        Get the top 20 meme IDs from Imgflip
    memegen        Generate a meme with a given meme template

    Files
    Command       Description
    del                Delete a file
    write                Write to a file
    read                Read the text content of a file
    ls                List the entire directory content
    get                Download a file striaght to the bot server

    Random
    Command       Description
    random           Generate a random number (buggy)
    fortnite           Get Fortnite news from all game modes, and the current Battle Royale map
    secret           sEcReT!?! 😳

    General
    Command       Description
    help                The... help?
    help.                Get more info on a command (prefix+help.command)
    ping                Ping the bot and get the bot latency.
    hi                Greets you in a Discord-friendly way - hi :)

    Moderation
    Command       Description
    ban                Ban a user. 100% satisfaction
