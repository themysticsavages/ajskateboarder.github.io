Commands

All the commands Cloudbot has to offer (as of now)

😐 General

Command Description
help The... help?
hi Noob version of ping
cclear Clear the console
clear Clear 99 messages
ping Receive the API latency
uptime Receives the uptime in seconds
poll Start a poll - customizable

👌 Utilities

Command Description
search Search on Yahoo for something
weather Receive the weather at a specific location
gif Get a GIF from Giphy
scratch Get info on a scratch user
youtube Receive 5 YouTube URLs
shorten Shorten a URL
download Download a file to the bot servers
godaddy Get domain availability
rickroll Check if a link is a rickroll

📁 File-server

Command Description
write Write to a file (if you have perms)
read Read the content of a file
del Delete a file
ls List all the present files (may be full)

❓ Random

Command Description
random Generate a unsecure number
endecode Encode text to ASCII and back (somewhat secure)
fortnite Get news and the map from the worst game, Fortnite
secret sEcReT?!? 😳

🔧 Moderation

Command Description
ban Ban a member - 100% satisfaction

🤑 Economy

Command Description
shop/add Register yourself as a user
shop/remove Unregister yourself from the bot
shop/money Receive 8-12 dollars
shop/buy Buy an item from the store!

🎵 Music

Command Description
play Play music (autoplay when finished)
end End any music currently playing
