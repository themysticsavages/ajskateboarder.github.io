This presentation:


is made almost completely from QuikImpakt. It summarizes drug information as visual data, as the slides show. This slideshow summarizes information about Aspirin.

How does it work?

QuikImpakt is made with Python 3 and the Flask framework, which powers many other large websites, like Netflix or YouTube.

The following APIs were used:

  • pip: collections
  • pip: pillow
  • pip: beautifulsoup4
  • pip: summarizer4u
  • photoroom
